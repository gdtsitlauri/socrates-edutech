# Linear Equations

Prerequisite: Students should understand Fractions and basic Numeracy.

Learning Objective: Solve one-variable equations and explain each transformation.

Equation: $2x + 3 = 11$

Definition: An equation states that two expressions are equal.

## Worked Example

1. Subtract 3 from both sides.
2. Divide both sides by 2.
3. Conclude that $x = 4$.
